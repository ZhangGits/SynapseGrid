### 1.1 Verification Agent 可信度评估（3 天）

**文件**：`app/agents/verification.py`

**改造内容**：
- Prompt 从"检测矛盾"改为"独立评估每个 finding 的可信度"
- 评估四维度：`evidence_strength`、`source_reliability`、`reasoning_soundness`、`data_consistency`
- 输出 `CredibilityAssessment` 数组，不再以 `conflicts` 为核心

```python
class CredibilityAssessment(BaseModel):
    finding_id: str
    evidence_strength: float      # 0-1
    source_reliability: float   # 0-1
    reasoning_soundness: float  # 0-1
    data_consistency: float     # 0-1
```

### 2.1 数据模型扩展（2 天）

```python
class LogicNode(BaseModel):
    id: str
    type: str  # "question" | "conclusion" | "claim" | "evidence"
```

```python
class PostProcessor:
```

### 2.4 Synthesis Agent 结构化翻译（3 天）

**文件**：`app/agents/synthesis.py`

**新职责**：
```python
async def run(self, raw_markdown: str, validation: ValidationResult) -> SynthesisResult:
    # 1. 解析 <finding> 标签
    findings = self._parse_finding_tags(raw_markdown)
    
    # 2. 调用 LLM 提取论证结构 → LogicGraph JSON
    logic_graph = await self._build_logic_graph(raw_markdown, findings, validation)
    
    # 3. 复用 PostProcessor
    enriched = self.post_processor.enrich(raw_markdown, validation)
    
    return SynthesisResult(
        logic_graph=logic_graph,
        enriched_markdown=enriched,
        node_map=self._build_node_map(raw_markdown, findings)
    )
```

**LLM Prompt（结构提取）**：
> "请从以下 Markdown 报告中提取论证结构，返回 JSON：节点列表（question/conclusion/claim/evidence）和边列表（supports/perspective_difference/tension/genuine_contradiction）。"

**验收**：LogicGraph 节点数 = finding 数；边类型正确；JSON Schema 校验通过。

### 2.5 Orchestrator 双路径分支（2 天）

**文件**：`app/core/orchestrator.py`

```python
async def run_task(self, request: TaskRequest) -> TaskResponse:
    ctx = await self._create_session(request)
    raw_markdown, findings = await self._run_research(ctx, request)
    validation = await self._run_verification(ctx, findings, request)
    
    if request.output_mode == "markdown":
        enriched = self.post_processor.enrich(raw_markdown, validation)
        await self._append_markdown_generated(ctx, enriched, validation)
        return TaskResponse(
            markdown_content=raw_markdown,
            enriched_markdown=enriched,
            canvas_schema=None,
            ...
        )
    else:
        synthesis = await self._run_synthesis(ctx, raw_markdown, validation, request)
        await self._append_synthesis_generated(ctx, synthesis)
        return TaskResponse(
            markdown_content=raw_markdown,
            enriched_markdown=synthesis.enriched_markdown,
            canvas_schema=synthesis.logic_graph,
            ...
        )
```

**验收**：Markdown 模式无 `SynthesisGenerated` 事件；Canvas 模式有 `LogicGraph`。
