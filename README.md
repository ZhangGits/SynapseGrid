# SynapseGrid

可审计的事件溯源 AI Agent 系统。

## 设计目标

解决传统 LLM 应用**不可审计、不可回退、不可验证**的痛点。基于事件溯源架构，将每个 Agent 的分析结果记录为不可变事件，支持完整审计追踪、分析要点回退和内容完整性验证。

## 快速开始

```bash
# Docker 启动
docker-compose up -d
# 访问 http://localhost:80
```

## 功能

- **三阶段 Agent 流水线**：Research → Verification → Synthesis，每阶段可独立配置 LLM 模型
- **事件溯源审计**：所有操作记录为不可变事件，支持完整回放和 Merkle 树完整性验证
- **回退机制**：可回退任意分析要点，基于剩余 Findings 重新综合报告
- **双模式输出**：Markdown 模式 / 画布可视化模式
- **成本追踪**：每个 Agent 独立记录 Token 消耗和费用

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/tasks` | 创建分析任务 |
| POST | `/api/v1/tasks/{id}/rollback` | 回退分析要点 |
| POST | `/api/v1/tasks/{id}/feedback` | 画布反馈 |
| POST | `/api/v1/tasks/{id}/resynthesize` | 重新综合 |
| POST | `/api/v1/verify` | 验证内容完整性 |
